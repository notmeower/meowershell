How to build meowershell WITH the build script?

1. Clone https://github.com/notmeower/meowershell.git
2. Add execution privileges to the build script
3. Execute the build script
4. The shell binary is in target/release: meowershell-loader

How to build meowershell WITHOUT the build script?
1. Clone https://github.com/notmeower/meowershell.git
2. In the project root, run cargo build --release -p meowershell-loader
3. The shell binary is in target/release: meowershell-loader
